## 📝 후기 및 팀 노션 페이지

<code>용주성</code> 좋은 팀원분들을 만나서 소통하기 너무 편안했고, 서로 많이 배려해주셔서 협업 과정이 힘들지 않았습니다. 제게 부족한 부분을 팀원분들께서 채워주시고 많이 배울 수 있는 과정이였습니다.

<code>김현지</code> 프론트 분들과 두 번째 협업이었는데 좋은 팀원분들을 만나서 너무 좋았습니다. 서로 많이 배려하면서 소통하면서 작업을 해 나간 덕분에 협업 과정이 힘들지 않았고, 그 덕분에 좋은 결과물이 나올 수 있지 않았나 싶습니다. 다른 백엔드 팀원분에게도 많이 배울 수 있는 시간이었고 좋은 시간이었습니다.
